Developer

Seorang developer adalah arsitek digital yang menciptakan dunia di mana ide-ide menjadi kenyataan. Mereka adalah pemikir kreatif yang menghadirkan konsep-konsep yang inovatif dan menjadikannya hidup melalui pemrograman dan desain. Dengan kombinasi keterampilan teknis dan imajinasi, mereka adalah pembangun jembatan antara ide dan realitas digital.

Desain Grafis

Desain grafis adalah seni visual untuk menyampaikan ide, pesan, atau informasi menggunakan elemen-elemen seperti gambar, teks, dan warna. Para desainer grafis menggabungkan kreativitas mereka dengan keterampilan teknis untuk membuat komunikasi visual yang efektif, mulai dari logo dan brosur hingga situs web dan kampanye periklanan. Ini adalah proses yang memadukan estetika dengan fungsionalitas untuk menciptakan pengalaman visual yang memikat dan informatif.

Seni Teater

Seni teater adalah bentuk seni pertunjukan yang melibatkan pementasan langsung di depan penonton. Ini meliputi segala hal dari drama panggung tradisional hingga pertunjukan musikal, komedi, tragedi, dan eksperimental. Dalam seni teater, para aktor menggunakan dialog, gerakan, dan ekspresi wajah untuk menghidupkan karakter dan menceritakan cerita. Ini adalah bentuk seni kolaboratif di mana penulis, sutradara, aktor, desainer set, dan kru lainnya bekerja bersama untuk menciptakan pengalaman yang menginspirasi dan menghibur penonton.

Web Developer
Kisah Perjalanan Saya Menjadi Web Developer

Nama saya Mohammad Humam Azka ReformasiYanto, seorang sarjana S1 jurusan Sistem Informasi dari [Nama Universitas Anda]. Sejak kuliah, saya sudah sering membuat proyek berbasis web. Melalui proyek-proyek ini, saya mengembangkan keahlian dan semangat dalam bidang pengembangan web.


Proyek-Proyek Utama

Proyek besar pertama yang saya buat adalah sebuah web admin absensi siswa di MA. Miftahul Huda Jleper. Dalam proyek ini, saya membangun sistem yang memungkinkan pengelolaan data absensi siswa secara efisien dan terintegrasi. Proyek ini menggunakan bahasa pemrograman PHP, HTML, CSS, dan JavaScript, serta memanfaatkan SQL untuk pengelolaan database.
Selanjutnya, saya membuat sebuah web pemesanan makanan dan minuman untuk sebuah restoran dengan membuat simulasi nama resto sendiri. Proyek ini tidak hanya berfungsi sebagai platform pemesanan, tetapi juga sebagai alat manajemen menu dan inventaris. Pengalaman ini mengajarkan saya tentang pentingnya user experience dan responsivitas dalam aplikasi web.
Saya juga mengembangkan sebuah web profil untuk komunitas Keluarga Segitiga Teater Kudus. Proyek ini bertujuan untuk menyediakan platform bagi anggota komunitas untuk berbagi informasi, kegiatan, dan pencapaian mereka. Saya menggunakan Firebase sebagai backend untuk pengelolaan data real-time dan autentikasi pengguna.
Selain proyek-proyek besar tersebut, saya juga telah mengerjakan berbagai web lainnya yang tidak saya sebutkan. Setiap proyek memberikan tantangan dan pelajaran baru yang memperkaya pengetahuan dan keterampilan saya dalam pengembangan web.

Android Developer
Proyek Besar Sebagai Android Developer

Sebagai Android developer, saya juga sering membuat proyek aplikasi menggunakan bahasa Java selama kuliah. Aplikasi ini memungkinkan guru untuk mengambil absensi siswa dengan memanfaatkan teknologi geolokasi dan foto selfie, memastikan keakuratan dan validitas data kehadiran siswa.
